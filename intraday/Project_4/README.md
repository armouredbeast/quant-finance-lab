Intraday Breakout Strategy (prototype)
-------------------------------------
Notes:
  - yfinance provides limited intraday data. For robust intraday, use a broker/data API (Alpaca, Polygon).
How to run:
  pip install pandas numpy yfinance matplotlib
  python intraday_breakout.py

# PILLAR 1 — PROJECT 3  
## Distribution Explorer (PDF + CDF Visualizer)

This project builds intuition for probability distributions by visualizing:

- Empirical PDF / PMF
- Empirical CDF
- Shape, skewness, tails, and spread

### Included distributions
**Discrete**
- Bernoulli(p)
- Binomial(n, p)
- Poisson(λ)

**Continuous**
- Uniform(a,b)
- Exponential(λ)
- Normal(μ,σ)
